# Build APK Without Android Studio

Use GitHub Actions to build the APK online.

1. Create a free GitHub account if you do not have one.
2. Create a new repository.
3. Upload all files from this `prism-calculator-android` folder to the repository.
4. Open the repository on GitHub.
5. Go to the `Actions` tab.
6. Select `Build Android APK`.
7. Click `Run workflow`.
8. Wait for it to finish.
9. Open the finished run and download the artifact named `prism-calculator-apk`.
10. Unzip it. Inside will be `app-debug.apk`.

To install on your phone:

1. Send `app-debug.apk` to your Android phone.
2. Open the file on the phone.
3. Allow `Install unknown apps` if Android asks.
4. Tap `Install`.
